# Streamlit NLP Pipeline Demo

This is a simple Natural Language Processing (NLP) pipeline demonstration built with Streamlit. The application provides two main features:
1. Translation: Automatically detects the source language and translates the text into a selected target language using `deep-translator`.
2. Spell Correction: Detects the language of the input text and corrects typographical errors using `pyspellchecker` and `nltk`.

## Project Structure

- `app.py`: The main Streamlit application containing the user interface and logic for both translation and spell checking.
- `pyproject.toml`: Configuration file defining project metadata and dependencies.
- `uv.lock`: Lockfile generated by `uv` to ensure reproducible virtual environments.
- `README.md`: Documentation for the project.

## Installation and Running Locally

This project uses `uv` for fast dependency management.

### Prerequisites

Ensure you have `uv` installed on your system. If not, install it using one of the following methods:

- Windows (PowerShell):
  ```bash
  powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
  ```
- macOS/Linux:
  ```bash
  curl -LsSf https://astral.sh/uv/install.sh | sh
  ```
- Or via pip:
  ```bash
  pip install uv
  ```

### Setup and Run

1. Navigate to the project directory in your terminal.
2. Install dependencies and set up the virtual environment by running:
   ```bash
   uv sync
   ```
3. Start the Streamlit application:
   ```bash
   uv run streamlit run app.py
   ```

The application will automatically open in your default web browser at `http://localhost:8501`.
